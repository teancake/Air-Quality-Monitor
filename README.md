# Air-Quality-Monitor
Schematic and PCB design of an air quality monitor.

### Description ###
This WiFi switch module utilises the [ESP-12E/F adaptor](https://github.com/teancake/ESP8266-ESP12-Adaptor "ESP-12E/F Adaptor").
